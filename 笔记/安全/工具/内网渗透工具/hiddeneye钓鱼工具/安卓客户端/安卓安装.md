# hiddeneye安装

## TERMUX App安装

从PlayStore安装{Termux}，然后运行下列命令：
```
pkg install git python 
```
```
php curl openssh grep
```
```
pip3 install wget
```
```
git clone -b Termux-Support-Branch https://github.com/DarkSecDevelopers/HiddenEye.git
```
```
chmod 777 HiddenEye
```
```
cd HiddenEye
```
```
python HiddenEye.py
```
